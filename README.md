# ccs8-mongodb-react-ping
It contains Mondb connection to react app
Step 1: Set up MongoDB Database
Step 2:Create Project Folder in Visual Studio Code [CCS8-mongodb-react-ping]
        1) Set Up the Backend (Express Server)
           mkdir backend
           cd backend
           Directory structure : CCS8-mongodb-react-ping -> backend
        2) Initialize a Node.js project and install required packages:
           npm init -y
           npm install express mongoose dotenv cors body-parser
